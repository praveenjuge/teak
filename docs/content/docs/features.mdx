## Card System

Eight card types cover common design workflows:

- <FileText size={16} className="inline" /> **Design Notes** — project decisions, feedback
- <Link size={16} className="inline" /> **Design Resources** — links with auto-metadata & screenshots
- <Camera size={16} className="inline" /> **Visual References** — screenshots, mockups, mood boards
- <Video size={16} className="inline" /> **Animation References** — motion examples with player
- <Mic size={16} className="inline" /> **Voice Notes** — quick ideas & client feedback
- <File size={16} className="inline" /> **Design Files** — systems, guidelines, assets
- <Palette size={16} className="inline" /> **Color Palettes** — extracted from inspiration
- <Quote size={16} className="inline" /> **Quote Cards** — highlighted quotes with context

## Core Capabilities

- Real-time sync across web, mobile, and extension
- Visual search with previews and design filters
- Masonry grid for browsing inspiration
- Secure storage with thumbnails and previews
- AI enrichment: auto-tags, summaries, categorization
- Admin dashboard for pipeline monitoring
- Feedback system across platforms
- Subscription management via Polar

## Platforms

- <Globe size={16} className="inline" /> **Web App** — full dashboard for bookmarking
- <Smartphone size={16} className="inline" /> **Mobile** — capture inspiration on the go
- <Plug size={16} className="inline" /> **Extension** — save from any website instantly

## Who It's For

Designers, developers, and creative teams who need to capture, organize, and search visual inspiration efficiently.

This update makes Teak more powerful across web and mobile. You get full card creation and management, smarter layouts, an initial Chrome extension with auto-save and context menu capture, plus subscriptions and upgrade prompts to manage usage cleanly.

- ✨ **New Feature**: Smarter masonry grid with bulk actions and better card layout.
- ✨ **New Feature**: Initial Chrome extension with auto-save, context menus, and improved popup UX.
- ✨ **New Feature**: New Subscription page and upgrade prompt when card limits are reached.

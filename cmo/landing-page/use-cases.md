# Use Cases

*How people use Teak in real life.*

## Primary Use Cases

### 1. Capture Ideas on the Go

**User story:** I find something interesting and want to save it for later.

**How:** Browser extension, mobile app

**Outcome:** Never lose a good idea again

---

### 2. Research Projects

**User story:** I'm researching a topic and need to collect and organize sources.

**How:** Tags, search, masonry grid

**Outcome:** All my research in one place, easily searchable

---

### 3. Creative Inspiration

**User story:** As a creator, I collect inspiration for future work.

**How:** Visual grid, images, links

**Outcome:** Visual inspiration board I can revisit

---

### 4. Knowledge Base

**User story:** I want to build a personal knowledge base I can reference.

**How:** Cards, search, filters

**Outcome:** My second brain

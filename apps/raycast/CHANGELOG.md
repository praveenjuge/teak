# Changelog

## 0.0.2 - 2026-02-11

- Switched linting to Raycast's canonical lint flow (`ray lint`) and added publish script (`ray publish`).
- Improved command metadata and copy for Raycast Store readiness.
- Hardened API error handling with user-facing error messaging and remediation hints.
- Added retry actions and standardized empty/error states in `Search Cards` and `Favorites`.
- Added stable backend error codes for Raycast HTTP endpoints and API-key rate limiting (429).
- Expanded Raycast and Convex tests for auth, validation, and error contract coverage.
- Added workspace release documentation and store listing draft copy.

## 0.0.1 - 2026-02-11

- Initial Raycast extension release with `Quick Save`, `Search Cards`, and `Favorites`.

Teak for Safari saves the page you are viewing in one click. The Chrome extension adds richer capture options for images, links, selected text, and supported social media posts.

## Install

The Chrome extension works in Chrome and any Chromium-based browser (Edge, Brave, Arc, etc.). Teak for Safari is a native macOS Safari extension app.

## Safari Extension

Teak for Safari is intentionally focused: clicking the Safari toolbar extension signs you in when needed and saves the current page URL to Teak when you are signed in. It does not add inline save buttons to pages.

## Auto-Save on Open

When you open either extension popup, Teak automatically saves the current page. The popup shows whether the page was saved, was already in your library, or cannot be saved.

## Chrome and Chromium Features

The context menu and inline save buttons below are available in Chrome and Chromium-based browsers. Teak for Safari stays focused on saving the current page from the Safari toolbar.

### Context Menu

Right-click anywhere on a page to access quick-save actions:

| Menu item | What it saves |
|---|---|
| **Save Page to Teak** | The current URL |
| **Save Text to Teak** | The selected text (creates a text card) |

Context menu saves are available on all standard web pages.

### Social Media Inline Save Buttons

On supported sites, Teak injects a save button directly into each post or story. You can save individual items without leaving the feed.

| Platform | Supported content |
|---|---|
| [instagram.com](https://www.instagram.com) | Posts and Reels |
| [pinterest.com](https://www.pinterest.com) | Pins |
| [x.com](https://x.com) | Posts (tweets) — saved with tweet text, author, and attached images or videos |
| [news.ycombinator.com](https://news.ycombinator.com) | Story links (saves the outbound article URL) |
| [sidebar.io](https://sidebar.io) | Story links |
| [webdesignernews.com](https://www.webdesignernews.com) | Story links |
| [heydesigner.com](https://heydesigner.com) | Story links |

Inline save buttons appear automatically once you are signed in to the extension. Each saved post is deduplicated — saving the same post twice will not create a duplicate card.

### Free Tier Limit

When you reach your free plan card limit, the Chrome extension popup shows an upgrade prompt instead of saving. Click **Upgrade to Pro** to open Teak Settings.

## Sign In

In Safari, choose **Sign in** to complete authentication in the Teak web app. In Chrome and Chromium-based browsers, the popup provides **Login** and **Create Account** actions that open the web app in a new tab.

## Troubleshooting

### Extension popup shows an error

- Check that you are signed in at [app.teakvault.com](https://app.teakvault.com).
- Reload the tab you are trying to save and try again.

### Chrome inline buttons do not appear

- Make sure you are signed in to the extension.
- Reload the page. Buttons are injected after the page fully loads.
- Verify the extension is enabled in `chrome://extensions`.
- Inline buttons are supported on Instagram, Pinterest, X (Twitter), Hacker News, Sidebar, Web Designer News, and HeyDesigner. They are not available on other sites.

---
title: "Instagram saves now persist post images and videos"
date: "2026-03-11"
---

- Instagram post and reel saves now persist attached media into Teak instead of saving only the page shell. Public image posts, videos, reels, and supported carousels render their downloaded media inline in the card modal, matching the existing X (Twitter) post experience more closely.
- When Instagram does not expose downloadable media, Teak now falls back cleanly to the existing link preview flow so the save still succeeds with the best available preview metadata.

---
title: "May 2026: desktop relaunch, Raycast, and reliability"
date: "2026-05-30"
---

- The macOS desktop app is rebuilt with real-time sync, browser-based sign-in, native menus, drag-and-drop uploads, and automatic updates, and now shows clear notifications with download progress while checking for and installing updates.
- Raycast gained one-shot saving, Save Selected Text, a tag browser with per-tag counts, and AI tools for recent cards, listing tags, and toggling favorites.
- Drag files anywhere in the web and desktop apps to upload in place, with one progress summary per batch, and the composer's Upload button shares the same flow; the add area is now a focused note with capture actions in the header.
- Mobile is more reliable: photos, videos, audio, and documents save successfully again, sign-in is restored on launch without bouncing to the welcome screen, appearance choices persist across restarts, and going offline shows a clear retry screen.
- Paging through your cards with the API now returns every card instead of sometimes stopping early on a full page.
- Link previews now share only a site's domain, never the full saved address, when loading site icons, so private links stay private.
- Filtering by a tag with spaces in Raycast now searches that exact tag instead of treating the extra words as a separate search.
- If you are on an older desktop build, update to the latest release before signing in again.

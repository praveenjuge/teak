---
title: "Native app sign-in foundation"
date: "2026-05-16"
---

- Desktop sign-in now uses the shared native app handoff, preparing the same secure browser-based login flow for future native Teak surfaces.
- Teak for Safari is now in foundation testing for Mac, with sign-in and one-tap page saving from the Safari toolbar.
- Older desktop builds may need to update before signing in again after this release.

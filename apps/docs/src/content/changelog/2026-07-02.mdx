title: "July 2026: sign in with your browser everywhere"
date: "2026-07-02"

- Raycast now lets you sign in with your browser in one step. Copying an API key is no longer required to get started, and existing keys keep working.
- Connect Teak to Claude and other MCP or API clients by signing in with your browser. API keys are still supported.
- Signing out of the desktop app no longer signs you out of Teak in your browser.

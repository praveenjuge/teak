## UI Rules

- The docs site uses Astro + Starlight with TailwindCSS.
- Use Starlight's built-in components and theming for documentation pages.
- Use TailwindCSS utility classes for custom components.
- Use `rounded-md` for all rounded corners.
- Don't use shadows as much as possible.
- Use `p-4` for most padding.
- Use `gap-4` for most gaps.

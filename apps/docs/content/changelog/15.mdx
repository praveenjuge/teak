date: "2026-02-11"
---

Teak now supports a dedicated Raycast extension with secure API key authentication, plus new settings and backend endpoints for keyboard-first capture and retrieval.

### Highlights

- ✨ **New Feature**: Raycast extension app with `Quick Save`, `Search Cards`, and `Favorites` commands
- ✨ **New Feature**: API key management in Teak settings (generate, copy-once, revoke)
- ✨ **New Feature**: New Convex HTTP endpoints for Raycast quick-save/search/favorites
- 💪 **Improvement**: Duplicate URL detection integrated into Raycast quick-save flow

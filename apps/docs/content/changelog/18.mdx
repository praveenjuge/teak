Teak Desktop is now ready for first public distribution on macOS Apple Silicon with a signed and notarized app bundle, direct GitHub Releases distribution, and built-in automatic updates.

### Highlights

- ✨ **New Feature**: macOS desktop release pipeline with tag-based GitHub automation.
- ✨ **New Feature**: Signed updater artifacts and metadata publishing for in-app updates.
- 💪 **Improvement**: Hardened desktop Tauri security policy with explicit production CSP.
- 💪 **Improvement**: Desktop docs now include install, update, rollback, and troubleshooting guidance.

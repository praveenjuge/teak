- 💪 **Improvement**: Raycast integration updated to use current API routes and response patterns
- 💪 **Improvement**: Incoming share flow on mobile hardened for more reliable cross-app imports
- ✨ **New Feature**: Theme preference storage and app-wide theme consistency on mobile
- ✨ **New Feature**: Card modal URL synchronization for deep-linking on web and desktop

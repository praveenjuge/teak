We’ve upgraded the login experience with Better Auth, bringing smoother sign‑in, sign‑up, and password recovery. New dashboard card modals let you edit cards directly, while AI metadata and Dribbgle categorization give you smarter content handling. All updates keep Teak intuitive and powerful.

### Highlights

- ✨ **New Feature**: Better Auth now powers sign‑in, sign‑up and password recovery
- ✨ **New Feature**: Dashboard cards can be edited via new modal components
- 💪 **Improvement**: AI metadata workflow added; tags must be single words
- ✨ **New Feature**: Dribbble added as a new categorization source

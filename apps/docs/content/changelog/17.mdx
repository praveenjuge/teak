Teak now exposes a remote MCP server on the API gateway, enabling MCP clients to create, search, update, favorite, and delete cards through the same backend logic as the public `v1` REST API.

### Highlights

- ✨ **New Feature**: Remote MCP server at `https://api.teakvault.com/mcp`
- 💪 **Improvement**: MCP tools execute through the same canonical `v1` gateway codepath for policy and behavior consistency
- 📚 **Docs**: Added dedicated MCP documentation with authentication, tool contracts, and JSON-RPC examples

---
title: "Raycast gets two new commands and AI tools"
date: "2026-03-09"
---

- **Save Clipboard** is a new Raycast command that saves your clipboard contents to Teak in one step — no form, no friction, just run the command.
- **Save Current Browser Tab** is a new Raycast command that saves the active tab directly from Raycast. Requires the [Raycast Browser Extension](https://www.raycast.com/browser-extension).
- The single card details view in Raycast was updated with a refreshed layout.
- **Raycast AI Tools**: The extension now exposes three tools for [Raycast AI](https://www.raycast.com/ai) — **Save to Teak** (save a note or URL), **Search Teak Cards** (search by text, type, tags, and more), and **Get Teak Card** (fetch full card details by ID). No extra setup: the tools reuse your existing API key.

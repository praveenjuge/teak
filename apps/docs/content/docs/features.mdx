## Content Types

**Six card types to capture everything:**
- <span style={{display: "inline-flex", alignItems: "center", gap: "0.5rem"}}><FileText size={16} /> **Text**</span> - Notes, ideas, thoughts
- <span style={{display: "inline-flex", alignItems: "center", gap: "0.5rem"}}><Link size={16} /> **Links**</span> - Bookmarks with auto metadata extraction
- <span style={{display: "inline-flex", alignItems: "center", gap: "0.5rem"}}><Camera size={16} /> **Images**</span> - Photos with drag-and-drop upload
- <span style={{display: "inline-flex", alignItems: "center", gap: "0.5rem"}}><Video size={16} /> **Videos**</span> - MP4, MOV, WebM with built-in player
- <span style={{display: "inline-flex", alignItems: "center", gap: "0.5rem"}}><Mic size={16} /> **Audio**</span> - Voice memos and audio files
- <span style={{display: "inline-flex", alignItems: "center", gap: "0.5rem"}}><File size={16} /> **Documents**</span> - PDFs and file attachments

## Core Capabilities

**Real-time sync** - Instant updates across web, mobile, and browser extension

**Smart search** - Full-text search with content type filtering and tagging

**Card system** - Beautiful masonry grid layout with soft delete and favorites

**File storage** - Secure uploads with automatic thumbnails via Convex Storage

## Platforms

- <span style={{display: "inline-flex", alignItems: "center", gap: "0.5rem"}}><Globe size={16} /> **Web App**</span> - Full-featured Next.js dashboard
- <span style={{display: "inline-flex", alignItems: "center", gap: "0.5rem"}}><Smartphone size={16} /> **Mobile**</span> - Native iOS/Android with Expo
- <span style={{display: "inline-flex", alignItems: "center", gap: "0.5rem"}}><Plug size={16} /> **Extension**</span> - Save content from any webpage

## Technology Stack

- **Backend** - Convex real-time database and serverless functions
- **Authentication** - Clerk with JWT integration
- **Frontend** - Next.js 15, React 19, TypeScript, Tailwind CSS
- **Mobile** - Expo React Native with TypeScript
- **Extension** - Wxt framework for Chrome

## Perfect For

Creative professionals, students, researchers, content creators, and knowledge workers who need a private, organized space to capture and rediscover information.

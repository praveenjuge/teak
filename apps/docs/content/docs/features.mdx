## Card System

Eight card types cover common design workflows. Each type is optimized for its content with automatic processing and rich previews.

### <FileText size={18} className="inline" /> Design Notes
Capture project decisions, meeting notes, and feedback. Full Markdown support with syntax highlighting for code snippets.

**Use cases:** Design rationale documentation, client feedback notes, quick ideas

### <Link size={18} className="inline" /> Design Resources
Save links with automatic metadata extraction and full-page screenshots. Teak captures the title, description, favicon, and a visual preview.

**Use cases:** Dribbble shots, CodePen demos, articles, documentation

### <Camera size={18} className="inline" /> Visual References
Store screenshots, mockups, and mood board images. Automatic color palette extraction helps you find images by dominant colors.

**Use cases:** UI screenshots, mood boards, reference photos, mockups

### <Video size={18} className="inline" /> Animation References
Save video files with an inline player and auto-generated thumbnails. Perfect for motion design references and tutorial clips.

**Use cases:** Micro-interactions, loading animations, tutorial segments

### <Mic size={18} className="inline" /> Voice Notes
Record audio directly or upload files. Automatic transcription makes voice notes fully searchable.

**Use cases:** Quick ideas on the go, verbal feedback, brainstorming sessions

### <File size={18} className="inline" /> Design Files
Upload PDFs, Sketch files, Figma exports, and other assets. Thumbnail previews help you browse without opening each file.

**Use cases:** Design systems, brand guidelines, exported assets

### <Palette size={18} className="inline" /> Color Palettes
Extract and save color palettes from any image. Each color includes hex values for easy copying.

**Use cases:** Brand colors, inspiration palettes, color research

### <Quote size={18} className="inline" /> Quote Cards
Save highlighted quotes with source context. Perfect for design principles, testimonials, and notable passages.

**Use cases:** Design principles, user quotes, inspirational text

## Search & Organization

### Smart Search

Type any word or phrase to search across card titles, descriptions, tags, transcriptions, and extracted text. Teak also recognizes special search tokens automatically:

| Token type | Examples | What it does |
|---|---|---|
| **Color name** | `red`, `blue`, `teal`, `purple`, `pink` | Filters cards by dominant palette color |
| **Visual style** | `minimal`, `dark`, `abstract`, `cinematic`, `pastel` | Filters cards by visual aesthetic |
| **Hex color** | `#FF5733`, `#3B82F6` | Filters cards by exact color match |
| **Date shorthand** | `today`, `this week`, `january`, `last year` | Filters cards by creation date |
| **Keyword** | anything else | Full-text match on card content |

Mix multiple tokens in one search — Teak applies all recognized filters simultaneously.

```
Examples:
- "blue button hover state"
- "minimal dark"
- "#3B82F6"
- "feedback from Sarah this week"
- "onboarding flow mobile"
```

### Smart Tagging
AI automatically analyzes your content and applies relevant tags. You can also add custom tags manually.

### Filtering
Filter by card type, favorites, visual style, color, or date. Active filters appear as removable pills in the search bar. Combine multiple filters to narrow down large collections quickly.

### Masonry Grid
Browse your inspiration in a Pinterest-style grid that adapts to different content sizes and aspect ratios.

## Cross-Platform Sync

Your inspiration syncs in real-time across all platforms:

### <Globe size={18} className="inline" /> Web App
Full-featured dashboard with keyboard shortcuts, bulk actions, and detailed card views. Best for extended browsing and organization sessions.

**Key features:**
- Drag-and-drop uploads
- Bulk selection and batch operations
- Full-screen card preview with metadata
- URL-synced card modal state for direct deep links

### <Monitor size={18} className="inline" /> Desktop App

Native macOS app with real-time sync and a full native menu bar. Download from [GitHub Releases](https://github.com/praveenjuge/teak/releases/latest) — no App Store required.

**Key features:**
- Signed and notarized macOS app (Apple Silicon, macOS 13+)
- Auto-updates on every launch with an interactive confirmation dialog
- Native macOS menu bar (About, Settings, Check for Updates, Log Out, Quit)
- Same real-time sync as web and mobile

See the [Desktop App guide](/docs/desktop) for install and troubleshooting details.

### <Smartphone size={18} className="inline" /> Mobile App

Native iOS app for capturing inspiration anywhere. Share directly from any app to save instantly.

**Key features:**
- Share sheet integration
- Incoming share import from other apps for links, text, and files
- Camera capture for quick screenshots
- Voice recording with one tap
- Offline viewing of synced cards

### <Puzzle size={18} className="inline" /> Browser Extension
One-click saving from any website. Right-click context menu for saving pages and selected text. Inline save buttons on social media feeds.

**Key features:**
- Auto-saves the current page when you open the popup
- Right-click any page to save it with full metadata
- Highlight text to save as a text card
- Inline save buttons on Instagram, Pinterest, and X (Twitter) posts

See the [Browser Extension guide](/docs/extension) for setup and full details.

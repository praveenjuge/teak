## Content Types

**Six card types for design workflows:**
- <span style={{display: "inline-flex", alignItems: "center", gap: "0.5rem"}}><FileText size={16} /> **Design Notes**</span> - Design decisions, feedback, project notes
- <span style={{display: "inline-flex", alignItems: "center", gap: "0.5rem"}}><Link size={16} /> **Design Resources**</span> - Inspiration links with auto metadata and previews
- <span style={{display: "inline-flex", alignItems: "center", gap: "0.5rem"}}><Camera size={16} /> **Visual References**</span> - Screenshots, mockups, mood boards with drag-and-drop
- <span style={{display: "inline-flex", alignItems: "center", gap: "0.5rem"}}><Video size={16} /> **Animation References**</span> - Motion design examples with built-in player
- <span style={{display: "inline-flex", alignItems: "center", gap: "0.5rem"}}><Mic size={16} /> **Voice Notes**</span> - Quick design ideas and client feedback recordings
- <span style={{display: "inline-flex", alignItems: "center", gap: "0.5rem"}}><File size={16} /> **Design Files**</span> - Design systems, brand guidelines, assets

## Core Capabilities

**Real-time sync** - Instant bookmark sync across web, mobile, and browser extension

**Visual search** - Smart search across bookmarked inspiration with visual previews and design-specific filtering

**Visual grid system** - Beautiful masonry layout perfect for browsing bookmarked design inspiration with favorites

**Secure file storage** - Automatic thumbnails and previews for all your design assets

## Platforms

- <span style={{display: "inline-flex", alignItems: "center", gap: "0.5rem"}}><Globe size={16} /> **Web App**</span> - Full-featured dashboard for visual bookmarking and inspiration management
- <span style={{display: "inline-flex", alignItems: "center", gap: "0.5rem"}}><Smartphone size={16} /> **Mobile**</span> - Capture inspiration on the go with native apps
- <span style={{display: "inline-flex", alignItems: "center", gap: "0.5rem"}}><Plug size={16} /> **Extension**</span> - Bookmark design inspiration from any website instantly

## Technology Stack

- **Backend** - Convex real-time database and serverless functions
- **Authentication** - Clerk with JWT integration
- **Frontend** - Next.js 15, React 19, TypeScript, Tailwind CSS
- **Mobile** - Expo React Native with TypeScript
- **Extension** - Wxt framework for Chrome

## Perfect For

Designers, developers, design teams, and creative professionals who need a visual bookmarking solution to capture design inspiration, organize creative ideas, and build comprehensive inspiration libraries.

## Card Types

**8 specialized card types for design workflows:**

- <FileText size={16} /> **Design Notes** - Project decisions, feedback
- <Link size={16} /> **Design Resources** - Links with auto-metadata &
  screenshots
- <Camera size={16} /> **Visual References** - Screenshots, mockups, mood boards
- <Video size={16} /> **Animation References** - Motion examples with player
- <Mic size={16} /> **Voice Notes** - Quick ideas & client feedback
- <File size={16} /> **Design Files** - Systems, guidelines, assets
- <Palette size={16} /> **Color Palettes** - Extracted from inspiration
- <Quote size={16} /> **Quote Cards** - Highlighted quotes with context

## Core Features

- **Real-time sync** across web, mobile, extension
- **Visual search** with previews & design filters
- **Masonry grid** layout for browsing inspiration
- **Secure storage** with thumbnails & previews
- **AI enrichment** - auto-tags, summaries, categorization
- **Admin dashboard** for pipeline monitoring
- **Feedback system** across platforms
- **Subscription management** via Polar

## Platforms

- <Globe size={16} /> **Web App** - Full dashboard for bookmarking
- <Smartphone size={16} /> **Mobile** - Capture inspiration on the go
- <Plug size={16} /> **Extension** - Save from any website instantly

## Who It's For

Designers, developers, and creative teams who need to capture, organize, and search visual inspiration efficiently.

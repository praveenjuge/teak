## Common Issues

### Browser Extension Issues

- Make sure your browser is up to date
- Try disabling other extensions temporarily
- Clear your browser cache and cookies

### Mobile App Issues

- Check that you're running the latest version of the app
- Try restarting the app
- Make sure you have a stable internet connection

## Contact Us

### Email Support

For technical issues and general questions:

- Email us at hi@praveenjuge.com
- We typically respond within 24-48 hours

### Bug Reports

If you've found a bug, please report it on GitHub:

1. Visit [GitHub Issues](https://github.com/praveenjuge/teak/issues)
2. Include steps to reproduce the issue
3. Describe what you expected to happen
4. Include your browser/OS version if relevant
5. Use the provided issue templates when available

### Feature Requests

We'd love to hear your ideas for new features:

- Submit feature requests on [GitHub Issues](https://github.com/praveenjuge/teak/issues)
- Use the "enhancement" label when creating new feature requests

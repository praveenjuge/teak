## Prerequisites

- **Docker & Docker Compose**
- **Bun 1.0+**

## Quick Start

```bash
git clone git@github.com:praveenjuge/teak.git
cd teak
bun run bootstrap
```

This will install dependencies, setup the database, and initialize environment files.

## Project Structure

- **Backend**: Hono.js API (`apps/backend`)
- **Web**: React + Vite (`apps/web`)
- **Mobile**: React Native + Expo (`apps/mobile`)
- **Database**: PostgreSQL + Drizzle ORM

## Development

```bash
# Start all services
bun run dev

# Individual services
bun run dev:backend    # API server
bun run dev:frontend   # Web app
bun run dev:mobile     # Mobile app
```

Access your apps:
- Web: http://localhost:3000
- API: http://localhost:3001

## Database

```bash
cd apps/backend

bun run db:studio      # Database GUI
bun run db:generate    # Create migrations
bun run db:migrate     # Apply migrations
bun run db:seed        # Add sample data
```

## Environment Setup

Copy the environment template:

```bash
cp .env.example .env
```

Key variables:
- `BETTER_AUTH_SECRET` - Auth secret key (required, 32+ chars)
- `POSTGRES_DB` - Database name (default: `teak_db`)

## Docker Commands

```bash
bun run docker:dev:down    # Stop containers
bun run docker:dev:clean   # Reset everything
```

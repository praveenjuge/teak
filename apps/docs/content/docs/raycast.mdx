Use Teak in Raycast to save ideas and find cards without leaving your keyboard flow.

## Quick setup

1. In Teak, open `/settings` > **API Keys**, then generate and copy a key.
2. Install the extension:

[![Install teak-raycast Raycast Extension](https://www.raycast.com/praveenjuge/teak-raycast/install_button@2x.png?v=1.1)](https://www.raycast.com/praveenjuge/teak-raycast)

3. In Raycast extension preferences, paste your Teak `API Key`.

## Commands

- **Quick Save**: Save clipboard text and links to Teak.
- **Search Cards**: Find and open cards instantly.
- **Favorites**: Jump to your starred cards.

## Troubleshooting

| Error | What it means | How to fix |
|---|---|---|
| **Set your API key** | No API key configured | Open Raycast extension preferences and paste your Teak API key |
| **Invalid or revoked key** | The key is wrong or was regenerated | Go to Teak Settings → API Keys and paste a fresh key |
| **Too many requests** | Rate limit hit | Wait a few seconds, then retry |
| **Unable to reach Teak** | Network connectivity issue | Check your internet connection and retry |
| **Card not found** | Card was deleted | Refresh the card list (`⌘R`) and try again |
| **Temporarily unavailable** | Teak backend error | Try again in a moment; if it persists, check [status.teakvault.com](https://status.teakvault.com) |

## Security

- Your API key only connects Raycast to your Teak account.
- Revoke or regenerate it anytime from Teak Settings.

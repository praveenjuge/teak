Use Teak in Raycast to quickly save text or links, search all cards, and jump to favorites without leaving your keyboard flow.

## Requirements

- Raycast installed on macOS
- Teak account
- One active Teak API key

## 1) Generate an API key in Teak

1. Open Teak Settings: `/settings`
2. Go to **API Keys**
3. Click **Generate API Key**
4. Copy the key immediately (it is shown once)

## 2) Install the extension in Raycast

1. Open Raycast
2. Open the Store listing: [Raycast Store search](https://www.raycast.com/store?query=teak)
3. Search for **Teak**
4. Install the extension

## 3) Configure extension preferences

Set the following in Raycast extension settings:

- `API Key`: key generated in Teak settings

## Publishing status

## Commands

### Quick Save

- Save ideas, links, and quick notes without leaving your keyboard flow.
- Use it when you find something worth revisiting and want it in Teak instantly.

### Search Cards

- Find saved inspiration in seconds with natural keywords.
- Open the result directly in Teak or jump back to the original source.

### Favorites

- Keep your most-used references one command away.
- Great for moodboards, reusable patterns, and current project references.

## Privacy

- Your API key only connects Raycast to your Teak account.
- You can revoke and regenerate it anytime from Teak Settings.
